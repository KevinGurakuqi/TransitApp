🔧 Google Maps API Setup Guide

If you see "Oops! Something went wrong" in the map:

This usually means the Maps JavaScript API is not enabled for your API key.

How to Fix:

  1. Go to Google Cloud Console:
    https://console.cloud.google.com/apis/library
  2. Search for "Maps JavaScript API" and click on it
  3. Click "Enable" if it's not already enabled
  4. Also enable "Directions API" (search for it and enable)
  5. Wait 1-2 minutes for the changes to propagate
  6. Refresh your website

Required APIs:

  • ✅ Maps JavaScript API (for displaying the map)
  • ✅ Directions API (for showing routes)
  • ✅ Geocoding API (already enabled - for address lookup)
  • ✅ Distance Matrix API (already enabled - for route calculations)

Check Your API Key Restrictions:

If you have API key restrictions enabled, make sure:

  • Your website domain is allowed (or use no restrictions for testing)
  • The APIs listed above are allowed

✅ After enabling the APIs:

Refresh your browser and try searching for a route again. The map should work!
